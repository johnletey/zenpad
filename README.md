# ZenPad

As _zen_ as it gets

[ZenPad](http://zenpad.now.sh) is a minimal web app for taking quick notes.

## Features:

- Ligatures
- Persistence
- Save to file
- Auto Indentation
- Syntax Highlighting
- GitHub Flavoured Markdown
